# CBRN Laboratory Intelligence Atlas — Master Demo Package

This package contains all demo versions in one GitHub Pages-ready structure.

## Folders
- `starter/`
- `v1/`
- `v2/`
- `v3/`
- `v4/`
- `v4.1/`
- `v4.2/`
- `v4.3/`

## Local run
```bash
python -m http.server 9090
```
Open:
```text
http://localhost:9090
```

## GitHub Pages
Upload all files and folders in this package to the root of one GitHub repository, then enable Pages from the `main` branch and `/root` folder.
