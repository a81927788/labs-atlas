# Global CBRN Defense Laboratory Benchmark Atlas

Ready-to-upload GitHub Pages starter template.

## What is included
- Dark intelligence-dashboard landing page
- Scientific methodology section
- Interactive Leaflet world map with dots for all labs
- Filters by country, region, domain and search text
- Country grouping pages
- Individual lab profile pages
- `data/labs.json` with 139 seed candidate labs/institutions

## Deploy on GitHub Pages
1. Create a GitHub repository, for example `cbrn-lab-atlas`.
2. Upload the contents of this folder.
3. Go to **Settings > Pages**.
4. Choose **Deploy from branch**, branch **main**, folder **/root**.
5. Open `https://YOUR_USERNAME.github.io/cbrn-lab-atlas/`.

## Important
The included dataset is a structured starter seed. Before official/executive use, validate every lab profile using official institutional sources, public reports, standards/accreditation records, and peer-reviewed publications.
